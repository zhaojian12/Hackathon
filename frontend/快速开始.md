# 快速开始指南

## 🚀 5 分钟快速上手

### 第一步：安装依赖

```bash
cd Hackathon/frontend
npm install
```

### 第二步：启动应用

```bash
npm run dev
```

应用将在 http://localhost:5173 启动

### 第三步：安装钱包（三选一）

#### 方案 A：MetaMask（狐狸钱包，推荐）
1. 访问 https://metamask.io/
2. 点击 "Download" 下载浏览器扩展
3. 创建新钱包或导入现有钱包

#### 方案 B：OKX 钱包
1. 访问 https://www.okx.com/web3
2. 下载并安装浏览器扩展
3. 创建新钱包或导入现有钱包

#### 方案 C：Fluent 钱包
1. 访问 https://fluentwallet.com/
2. 下载并安装浏览器扩展
3. 创建新钱包或导入现有钱包

### 第四步：连接钱包

1. 打开应用 http://localhost:5173
2. 点击右上角的 "连接钱包" 按钮
3. 在钱包扩展弹窗中点击 "连接"
4. 如果提示切换网络，点击 "切换网络"

### 第五步：获取测试代币

#### MetaMask/OKX 用户
1. 访问水龙头：https://efaucet.confluxnetwork.org/
2. 输入你的钱包地址
3. 点击 "Claim" 领取测试 CFX

#### Fluent 用户
1. 访问水龙头：https://faucet.confluxnetwork.org/
2. 输入你的钱包地址
3. 点击 "Claim" 领取测试 CFX

## ✅ 验证连接成功

连接成功后，你应该看到：
- ✅ 右上角显示钱包地址（如：0x1234...5678）
- ✅ 显示 CFX 余额（如：10.0000 CFX）
- ✅ 显示钱包类型图标
  - 🦊 OKX = OKX 钱包
  - 🦊 MetaMask = MetaMask 钱包
  - 💧 Fluent = Fluent 钱包

## 🎯 开始使用功能

现在你可以：

### 1. 创建交易
- 在左侧面板填写交易信息
- 选择代币类型
- 输入金额和对方地址
- 点击 "创建交易"

### 2. 查看交易列表
- 右侧面板显示所有交易
- 查看交易状态（待处理、已完成、已取消）
- 查看交易详情

### 3. 管理交易
- 接受交易：作为接收方确认交易
- 取消交易：作为创建者取消交易
- 完成交易：双方确认后完成交易

## 🔧 常见问题解答

### 问：钱包连接失败怎么办？
**答**：请检查以下几点：
- ✓ 钱包扩展是否已安装并启用
- ✓ 钱包是否已解锁
- ✓ 浏览器是否允许弹出窗口
- ✓ 刷新页面重试

### 问：为什么看不到余额？
**答**：
- 确认已连接到正确的测试网络
- 访问水龙头获取测试代币
- 等待几秒钟让余额更新
- 刷新页面重新连接钱包

### 问：同时安装了多个钱包怎么办？
**答**：
- 系统会自动选择优先级最高的钱包
- 优先级：OKX > MetaMask > Fluent
- 如需使用特定钱包，可临时禁用其他钱包扩展

### 问：如何切换钱包？
**答**：
- 在钱包扩展中切换账户
- 刷新页面
- 重新点击 "连接钱包"

### 问：网络切换失败怎么办？
**答**：
- 手动在钱包中添加网络
- 使用以下配置：
  - **网络名称**：Conflux eSpace Testnet
  - **RPC URL**：https://evmtestnet.confluxrpc.com
  - **Chain ID**：71
  - **货币符号**：CFX
  - **区块浏览器**：https://evmtestnet.confluxscan.io

## 🐛 调试技巧

### 查看日志
1. 按 F12 打开浏览器开发者工具
2. 切换到 "Console" 标签
3. 查看日志信息：
   - 🔍 = 检测信息
   - ✅ = 成功操作
   - ❌ = 错误信息

### 常见错误代码
- **4001**: 用户拒绝连接（正常，无需担心）
- **4902**: 网络不存在（系统会自动添加）
- **-32002**: 请求待处理（请在钱包中确认）

## 💡 使用技巧

### 提高效率
1. **保持钱包解锁**：避免频繁输入密码
2. **添加书签**：快速访问应用
3. **使用快捷键**：F5 刷新页面

### 安全建议
1. ⚠️ 仅在测试网使用测试代币
2. ⚠️ 不要在测试钱包中存放真实资产
3. ⚠️ 妥善保管助记词和私钥
